# ESP32-CAM Face Detection

Go service that reads the ESP32-CAM MJPEG stream, runs face detection using GoCV (OpenCV), and re-serves the annotated stream over HTTP.

## How It Works

```
ESP32-CAM (/stream)
    └── Go pipeline reads MJPEG frames
            └── GoCV detects faces (Haar cascade)
                    └── Draws green rectangles on detected faces
                            └── Broadcasts annotated MJPEG to all connected browsers
```

## Requirements

| Tool | Version | Install |
|------|---------|---------|
| Go | 1.26.2+ | [go.dev/dl](https://go.dev/dl) |
| OpenCV | 4.x | `brew install opencv` |
| GoCV | v0.37.0 | `go mod tidy` (auto) |

## Quick Start

### 1. Install OpenCV (once)

```bash
brew install opencv
```

### 2. Fetch Go dependencies (once)

```bash
cd face-detection
make tidy
```

### 3. Run

```bash
make run CAM=http://192.168.x.x/stream
```

Replace `192.168.x.x` with the IP shown in the ESP32-CAM serial monitor.

### 4. Open in browser

```
http://localhost:8080
```

You will see the live annotated stream with green rectangles around detected faces and a face count below the video.

## Endpoints

| URL | Description |
|-----|-------------|
| `http://localhost:8080/` | HTML UI with live annotated stream |
| `http://localhost:8080/stream` | Raw annotated MJPEG stream |
| `http://localhost:8080/health` | JSON face count `{"faces": N}` |

## Makefile Targets

| Target | Description |
|--------|-------------|
| `make deps` | Install OpenCV via Homebrew (once) |
| `make tidy` | Fetch Go module dependencies |
| `make build` | Compile binary → `./face-detection` |
| `make run CAM=<url>` | Run the server |
| `make clean` | Remove compiled binary |

## Configuration Flags

| Flag | Default | Description |
|------|---------|-------------|
| `--cam` | *(required)* | ESP32-CAM MJPEG stream URL |
| `--addr` | `:8080` | HTTP listen address |
| `--cascade` | `/opt/homebrew/share/opencv4/haarcascades/haarcascade_frontalface_default.xml` | Haar cascade XML path |

Override any flag at runtime:

```bash
go run . \
  --cam http://192.168.x.x/stream \
  --addr :9090 \
  --cascade /path/to/haarcascade_frontalface_default.xml
```

## Performance Note

The ESP32-CAM with GC2145 sensor streams at ~8 fps QVGA (320×240). Face detection runs on the host PC via OpenCV — latency depends on your machine, typically under 10 ms per frame on modern hardware.

## Use with OpenCV (Python)

The `/stream` endpoint is a standard MJPEG stream and works directly with OpenCV:

```python
import cv2
cap = cv2.VideoCapture("http://localhost:8080/stream")
while True:
    ret, frame = cap.read()
    if not ret:
        break
    cv2.imshow("faces", frame)
    if cv2.waitKey(1) == ord('q'):
        break
```
